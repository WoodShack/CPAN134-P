About Me

I'm Scott Woodhouse, a self taught developer, enrolled in Humber Polytechnics computer programming course.
I'm currently entering my second semester and actively building skills through hands on learning.
I have always been interested in developing useful, optimized software.
Whether it's an internal tool or an app released to the public, I find it extremely rewarding to develop software that others find beneficial and user friendly.

I've always been drawn to software development, the ability to create something from nothing and share it with the world is exciting.
Create a useful app or an entertaining game and it could be downloaded by thousands of differnt people that you will never even meet.

I've developed many skillsets over the years.

  • Released a QT application written in C++ and QML
  • Utilized the Godot game engine and published my own app on both the Apple App store & The Google Play store
  • Developed native android applications written in Java
  • Created an internal tool using PHP, CSS, HTML & JavaScript that connects to the Epicor ERP API
  • Skilled in database planning and ERD (Entity Relationship Diagram) design
  • Many More...
